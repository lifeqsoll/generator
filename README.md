# Password Generator

Простой и безопасный генератор паролей в одном файле.

### Как это работает
1. **Настройка**: Укажите нужную длину и количество спецсимволов/цифр.
2. **Генерация**: Нажмите кнопку — скрипт создаст случайную комбинацию.
3. **Копирование**: Пароль сразу попадет в буфер обмена.
4. **Сохранение**: Браузер предложит запомнить пароль благодаря полю `autocomplete="new-password"`.

### Особенности
- **Индикатор**: Полоска под паролем показывает его надежность.
- **Темы**: Поддержка темного и светлого режимов.
- **Приватность**: Весь код работает только в вашем браузере, данные никуда не передаются.

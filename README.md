# Password and Nickname Generator

Веб-приложение для генерации безопасных паролей и стилизованных никнеймов. Работает полностью на стороне клиента.

## Функциональные возможности

### Генератор паролей
*   Настройка длины пароля (от 4 до 50 символов).
*   Выбор количества специальных символов, заглавных букв и цифр.
*   Визуальный индикатор надежности (Strength Bar), меняющий цвет в зависимости от сложности.
*   Автоматическое копирование результата в буфер обмена.

### Генератор никнеймов
*   Генерация на основе ключевого слова или из встроенной базы.
*   Использование тематических словарей: киберпанк, готика, аниме и эмо-стилистика.
*   Случайное добавление стилистических модификаторов и символов (декораций).
*   Автоматическое копирование ника при создании.

### Интерфейс и системные функции
*   Минималистичный дизайн с поддержкой светлой и темной тем.
*   Переключение между инструментами через систему вкладок.
*   Функция быстрой очистки текущего результата с экрана.
*   Отсутствие внешних зависимостей (чистый HTML, CSS и JavaScript).

# SafeGEN

Веб-приложение для генерации безопасных паролей, стилизованных никнеймов и управления временной почтой. Работает полностью на стороне клиента, обеспечивая максимальную приватность.

## Функциональные возможности

### Генератор паролей
* **Настройка сложности:** Установка общей длины, количества спецсимволов, заглавных букв и цифр.
* **Индикатор надежности:** Визуальный Strength Bar, динамически меняющий цвет в зависимости от сложности пароля.
* **Smart Copy:** Автоматическое копирование результата в буфер обмена сразу после генерации.

### Генератор никнеймов (Step-by-Step)
* **Двухэтапная логика:** 1. Сначала генерируется уникальный никнейм на одну строку.
    2. После генерации появляется возможность создать безопасный пароль специально для этого ника.
* **Стилизация:** Использование тематических словарей (Cyberpunk, Gothic, Anime, Space, Myth, Post-punk).
* **Модификаторы:** Поддержка функции «Random Case» (cYbEr) и автоматическое добавление декоративных элементов (`xX_ _Xx`, `†`, `666`).

### Временная почта (Mail Integration)
* **Быстрый доступ:** Прямая интеграция с сервисом Guerrilla Mail для анонимных регистраций.
* **Импорт почты:** Умная функция захвата адреса из буфера обмена кликом по полю результата для сохранения в историю.

### Система истории и экспорт
* **Локальное хранилище:** Автоматическое сохранение последних 15 операций в `localStorage`.
* **Менеджмент:** Возможность удаления отдельных записей или полной очистки списка.
* **Экспорт данных:** Выгрузка всей истории в текстовый файл (.txt) для локального хранения.

## Технические особенности
* **Zero Footprint:** Не требует сервера, баз данных или регистрации — всё работает в вашем браузере.
* **UI/UX:** Минималистичный адаптивный дизайн с поддержкой темной и светлой тем.
* **Чистый стек:** Написано на чистом HTML5, CSS3 и JavaScript без внешних зависимостей.
